# ReactTV

> React renderer for TV (WebOS, SmartTV)

- [ ] WebOS
- [ ] SmartTV

## CLI

```
$ react-tv init <app-type> <app-name>
$ cd app-name
$ react-tv dev # emulate on browser
$ react-tv pack # should pack app
$ react-tv run # should run on emulator
```
